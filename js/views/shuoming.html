使用说明

本项目采用 angularjs + bootstrap + nodejs + requirejs 模式，介绍封装图表指令的引入使用。

1.将对应的directive.js文件添加到directive文件夹, chartService.js文件添加到service文件夹下。

2.在main.js文件里的require.config的path里引入文件。

        require.config({
        path: {
            'duidieBarDirective': './directives/duidieBarDirective',
            'pinghuaLineDirective': './directives/pinghuaLineDirective',
            'solidPieDirective': './directives/solidPieDirective',
        }
            ... ...
        )

3.在mainDirective里注入依赖。

        define(['duidieBarDirective','pinghuaLineDirective','solidPieDirective'], function(){})

4.在html文件里使用指令标签。如需查看更多指令请点击"基础指令"~_~
